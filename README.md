# AI Layout Assistant — v0.4.1 PRO (SaaS Edition)

**AI Layout Assistant** — это профессиональный SaaS-инструмент для фронтенд-разработчиков и контент-менеджеров, который превращает сырые данные и документы в чистую, семантическую верстку по методологии БЭМ.

## 🚀 Новые возможности версии 0.4.1

Проект эволюционировал из простого скрипта в полноценную облачную платформу:

*   **☁️ Серверная история (MySQL)**: Все ваши генерации теперь сохраняются в базе данных. Доступ к проектам возможен с любого устройства.
*   **🔐 Система авторизации**: Полноценная регистрация и вход. Ваши данные и история версий защищены и доступны только вам.
*   **📂 Поддержка документов**: Прямая загрузка файлов **Word (.docx)**, PDF и PPTX. ИИ сам извлечет текст и превратит его в разметку.
*   **🧩 Конструктор секций (Stack)**: Возможность сохранять отдельные блоки в "стек" и собирать из них полноценные страницы.
*   **🕒 Профессиональная история**: Новое модальное окно с карточками версий, временем генерации и быстрым предпросмотром кода.
*   **✨ Премиальный UI**: Полностью обновленный светлый интерфейс в стиле Vibrant SaaS для комфортной длительной работы.

## 🛠 Технологический стек

*   **Frontend**: Vanilla JavaScript (ES6+), Modern CSS (Flexbox/Grid).
*   **Backend**: PHP 8.x (Session-based Auth).
*   **Database**: MySQL (History & User Management).
*   **AI Engine**: OpenRouter API (Universal Free Router).
*   **Parsers**: `mammoth.js` для Word, встроенные потоковые парсеры для AI.

---

## 🏗 Архитектура системы

1.  **Client-Side**: Обработка файлов, стриминг ответов ИИ и живое превью в изолированном `<iframe>`.
2.  **API Layer (`api.php`)**: Обеспечивает безопасную связь фронтенда с базой данных, управляет сессиями и правами доступа.
3.  **Data Layer**: MySQL хранит профили пользователей и дерево версий верстки.

## 💻 Развертывание на VPS (Fast VPS)

Проект оптимизирован для работы на любом стандартном хостинге с поддержкой PHP и MySQL.

1.  **Загрузка файлов**:
    Скопируйте все файлы из репозитория в корневую папку вашего сайта (`public_html` или аналогичную).

2.  **Настройка Базы Данных**:
    *   Создайте новую базу данных MySQL в панели управления хостингом.
    *   Импортируйте файл `database.sql` через phpMyAdmin.

3.  **Конфигурация Backend**:
    *   Откройте файл `api.php`.
    *   Укажите данные для подключения к вашей БД:
        ```php
        $db_host = 'localhost';
        $db_name = 'your_db_name';
        $db_user = 'your_db_user';
        $db_pass = 'your_db_password';
        ```

4.  **Права доступа**:
    Убедитесь, что сервер имеет права на чтение/запись в папку проекта для корректной работы сессий PHP.

## 🎯 Ключевые функции

*   **"Золотой стандарт"**: Обучите ИИ вашему стилю кода, просто вставив примеры своего идеального HTML/CSS в настройках.
*   **Адаптивные таблицы**: Строгие правила генерации гарантируют, что любые данные будут оформлены в чистые `<table>` с BEM-классами.
*   **SEO Микроразметка**: Автоматическая генерация JSON-LD схем (Article, Product, FAQ) для каждой страницы.
*   **Refinement Chat**: Возможность вносить правки в уже готовую верстку обычными словами (например: "сделай заголовок красным").
