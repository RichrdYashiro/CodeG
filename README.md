# AI Layout Assistant — v0.1

**AI Layout Assistant** — это интеллектуальный инструмент для фронтенд-разработчиков и дизайнеров, который превращает сырые текстовые описания в чистую, семантическую верстку по методологии БЭМ в реальном времени.

## 🚀 О проекте

### Какую проблему решает?
Создание однотипных блоков верстки (карточки, секции, формы) вручную занимает много времени. Существующие решения часто выдают "грязный" код без структуры. Наш продукт автоматизирует рутину, выдавая код, который СРАЗУ соответствует внутренним стандартам качества компании.

### Для кого?
- **Фронтенд-разработчики**: для быстрой прототипизации и генерации базовой структуры компонентов.
- **Контент-менеджеры**: для самостоятельного создания сложных блоков без глубокого знания HTML.
- **Дизайнеры**: для проверки того, как их текст ложится в реальную сетку.

### Почему AI необходим?
Традиционные генераторы работают по жестким шаблонам. AI (DeepSeek) способен понимать контекст текста, выделять логические акценты и адаптировать структуру под конкретное содержание, сохраняя при этом заданный стиль кода (БЭМ).

## 🛠 Технологический стек

- **Core**: Vanilla JavaScript (ES6+), HTML5, CSS3.
- **AI Engine**: DeepSeek Chat (через OpenRouter API).
- **Streaming**: Fetch API + ReadableStream для мгновенной отрисовки кода.
- **Styling**: Современный CSS с использованием Flexbox/Grid и переменных.
- **Storage**: Browser LocalStorage для сохранения конфигураций.

**Почему такой выбор?**
- Максимальная легкость: проект работает без сборщиков и тяжелых фреймворков.
- Скорость: прямая работа с API обеспечивает минимальные задержки.

## 🏗 Архитектура системы

Проект представляет собой Single Page Application (SPA), работающее полностью на стороне клиента:

1.  **UI Layer**: Интерактивный интерфейс с темной темой, разделенный на панель ввода, редактор паттернов и живое превью.
2.  **Streaming Engine**: Модуль в `script.js`, который управляет соединением с OpenRouter, обрабатывает входящие чанки и парсит их.
3.  **Code Parser**: Логика разделения потока на CSS и HTML блоки с помощью кастомных маркеров (`###CSS###`, `###HTML###`).
4.  **Sandbox Preview**: Изолированный `<iframe>`, который обновляет свое содержимое через `srcdoc` при каждом изменении кода.

## 💻 Локальное развертывание

1.  **Клонируйте репозиторий**:
    ```bash
    git clone https://github.com/your-repo/IIStartUp.git
    ```
2.  **Настройте окружение**:
    - Откройте `config.js`.
    - Укажите ваш `OPENROUTER_API_KEY` в объекте `window.ENV` или введите его позже в интерфейсе настроек.
3.  **Запустите проект**:
    - Проект можно запустить просто открыв `index.html` в браузере.
    - Для корректной работы всех функций (например, если появятся модули) рекомендуется использовать локальный сервер (например, расширение VS Code `Live Server` или `npx serve .`).

## 🎯 Как пользоваться

1.  **Настройка**: Нажмите на иконку шестеренки и введите свой API-ключ OpenRouter.
2.  **Паттерны**: В настройках можно задать "Золотой стандарт" — примеры вашего идеального HTML и CSS кода. ИИ будет копировать этот стиль.
3.  **Ввод**: Вставьте любой текст (например, "Описание услуги: облачный хостинг за 500р, 3 тарифа, кнопка купить") в левое текстовое поле.
4.  **Генерация**: Нажмите "Создать верстку".
5.  **Результат**: Вы увидите, как в реальном времени заполняются вкладки с кодом и обновляется окно предпросмотра справа.
6.  **Копирование**: Нажмите кнопку копирования, чтобы получить готовый блок (HTML + `<style>`).
